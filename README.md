MeetCauca
Descripción

MeetCauca es una plataforma de videoclases en tiempo real diseñada para facilitar la interacción entre profesores y estudiantes en entornos virtuales. El sistema permite la comunicación mediante audio y video, además de incorporar herramientas de participación y seguimiento del aprendizaje.

Funcionalidades Principales
Videollamadas en tiempo real.
Salas independientes.
Roles de Profesor y Estudiante.
Chat integrado.
Reacciones con emojis.
Modo Clase.
Cola de turnos para participación.
Encuestas rápidas.
Semáforo de comprensión.
Control de participantes por parte del profesor.
Tecnologías Utilizadas
HTML, CSS y JavaScript
Vite
Node.js
Express
Socket.io
WebRTC
Mediasoup
